# Adam Freedom Stack

A motivational + financial freedom visualization app built with Next.js 14.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Pages

- `/` - Home page with hero section and overview
- `/strategy` - Strategy page outlining the three pillars
- `/financials` - Financial projections and goals

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Lucide React (icons)
- Framer Motion (animations)
